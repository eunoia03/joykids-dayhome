# Joy Kids Dayhome website

This is a dependency-free static website made with plain HTML, CSS, and JavaScript. It is ready to publish on GitHub Pages and does not need a build step.

## Main updates to make

- Replace the phone number, email address, general area, and caretaker name in the HTML files.
- Edit the sample cards in `announcements.html` to add current notices.
- Replace gallery placeholder images with approved photos. Keep the existing filenames to avoid changing the HTML.
- Replace the caretaker and home placeholders on `index.html` with real images when available.
- Update colours in the variables at the top of `assets/css/style.css`.

## Preview locally

Open `index.html` in a web browser. All page links work with relative paths.

## Publish with GitHub Pages

1. Upload the contents of this folder to the root of a GitHub repository.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the branch containing the site and the `/ (root)` folder, then save.

## Privacy reminder

Get written parent or guardian permission before publishing identifiable photos of children. Remove location details that should remain private.
